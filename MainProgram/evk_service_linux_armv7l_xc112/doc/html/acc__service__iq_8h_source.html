71 /**
72  * @brief Get running average factor
73  *
74  * The running average factor is the factor of which the most recent sweep is weighed against previous sweeps.
75  * Valid range is between 0.0 and 1.0 where 0.0 means that no history is weighed in, i.e filtering is effectively disabled.
76  * A factor of 1.0 means that the most recent sweep has no effect on the result,
77  * which will result in that the first sweep is forever received as the result.
78  *
79  * @param[in] service_configuration The configuration to set the running average factor for
80  * @return Running average factor
81  */

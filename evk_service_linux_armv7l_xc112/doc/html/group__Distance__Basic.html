void acc_detector_distance_basic_destroy ( acc_detector_distance_basic_handle_t **  handle)

Destroy a distance basic detector.

The distance basic detector handle reference is set to NULL after destruction. If NULL is sent in, nothing happens.

Parameters
[in]handleThe detector handle to destroy, will be set to NULL
